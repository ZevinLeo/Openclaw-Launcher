# 🦞 OpenClaw Manager

**OpenClaw Manager** 是专为 [OpenClaw](https://openclaw.ai/) 设计的图形化管理工具，基于 Flutter 开发，提供一站式的安装、配置、监控和管理体验。

无论您是开发者还是普通用户，通过此启动器，无需记忆复杂的 CLI 指令，即可轻松管理您的 AI 算力网络节点。

---

## ✨ 核心功能

### 📊 Dashboard
- **三列布局**：核心监控 | 核心文件 | 版本管理
- **一键启动/停止**：点击按钮控制服务启停
- **实时监控**：显示端口、进程ID、运行时长、Tokens
- **核心文件管理**：快速编辑配置文件
- **版本管理**：检查更新、一键升级

### 🤖 AI 配置
- **模型管理**：添加、编辑、删除模型
- **参数配置**：别名、最大Tokens、温度
- **主模型设置**：一键设为主模型
- **认证配置**：API Key 管理

### 💬 消息渠道
- **多渠道支持**：WhatsApp、Telegram、飞书、Discord、Slack、iMessage、QQ Bot
- **可视化配置**：启用开关、凭证配置、语音合成/识别
- **即时启用**：实时开关渠道

### 🛠️ 技能管理
- **技能列表**：查看已安装技能
- **技能市场**：浏览安装新技能

### 📝 应用日志
- **实时日志**：实时显示后台输出
- **日志筛选**：按类型显示（INFO/ERROR/SUCCESS/CMD）
- **日志复制**：一键复制所有日志

### ⚙️ 设置
- **主题切换**：亮色/深色/自动跟随系统
- **开机自启动**：开机自动运行程序
- **核心管理**：安装/修复/卸载核心

### 🖥️ Windows 集成
- **窗口沉浸**：标题栏跟随主题变化
- **最小尺寸**：限制窗口最小尺寸确保完整显示

## 📖 使用方法

### 1. 安装运行

下载打包好的 `.exe` 文件，直接双击运行即可。

### 2. 配置使用

#### 首次配置
1. 启动软件，自动进入 Dashboard
2. 点击"启动服务"启动 OpenClaw 核心
3. 在 AI 配置页面添加您的模型
4. 在消息渠道页面配置所需渠道
5. 在设置页面开启开机自启动（可选）

#### 日常使用
1. 启动软件，自动启动服务
2. 在 Dashboard 监控运行状态
3. 通过消息渠道与 AI 对话

## 🔧 开发指南

### 从源码运行

```bash
# 安装依赖
flutter pub get

# 运行调试
flutter run
```

### 构建发布

```bash
# 构建 Windows exe
flutter build windows --release
```

生成的 exe 在 `build\windows\x64\release\bundle\` 目录。
